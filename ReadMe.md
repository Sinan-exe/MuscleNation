# 💪 MuscleNation

A responsive landing page for a gym brand built with React and Tailwind CSS.

🔗 **Live Demo** → [muscle-nation.vercel.app](https://muscle-nation.vercel.app)

---

## Features

- Hero section with CTA
- Why Choose Us stats (members, trainers, programs, awards)
- Programs showcase (Flex Muscle, Basic Yoga, Weight Lifting)
- Pricing plans (Starter, Popular, Advance)
- BMI Calculator
- Responsive design across all screen sizes
- Clean navigation with smooth scroll

---

## Tech Stack

- **HTML** — for the structure
- **vanilla CSS** — smooth styling
- **JavaScript** — adding functionality
- **Vercel** — deployment

---

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Sinan-exe/muscleNation.git

# Navigate into the project
cd muscleNation

# Install dependencies
npm install

# Start the dev server
npm run dev
```
